# LoMTRPG キャラクリシート

アップロードされた Light of Moon TRPG ルールブックを参照して作成した、HTML/CSS/JavaScriptだけで動くキャラクター作成補助サイトです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 「基本」タブで出身・得意分野を選び、「出身ダイスを振る」を押します。
3. 能力値、強化、装備、技能、光の種、事務所を入力します。
4. 画面下部で所有技能点、合計使用技能点、残技能点を確認します。
5. 「保存庫」タブでローカル保存、クラウド保存、GM/管理者への提出、共有コードの受け渡しができます。
6. 「出力」タブからTekey用データ、ココフォリア用コマJSON、チャットパレット、テキスト、JSONを保存できます。

## GitHub Pagesで公開する

このサイトはビルド不要の静的サイトです。`index.html`、`style.css`、`app.js` をそのままGitHub Pagesで公開できます。

1. GitHubに新しいリポジトリを作成します。
2. このフォルダのファイルをリポジトリ直下に配置してpushします。
3. GitHubの `Settings` -> `Pages` で、公開元を `Deploy from a branch`、ブランチを `main`、フォルダを `/root` に設定します。
4. 表示されたPages URLを開きます。

共有URL機能は、公開後のPages URLでもそのまま使えます。共有URLにはキャラクターデータが含まれるため、公開前に秘匿情報やGM専用メモが入っていないか確認してください。

## 保存庫と共有

- ローカルアカウントはこのブラウザ内の保存名です。端末ごとの下書き保存として使えます。
- クラウドアカウントはSupabase Authを使ったメール/パスワードログインです。
- 「クラウド保存」でログイン中アカウントのSupabase DBに現在のキャラクターを保存できます。
- 「GMへ提出」を押すと `submitted` が有効になり、Supabase側で `admin` 権限を持つアカウントの管理者一覧に表示されます。
- 管理者一覧から提出キャラクターを確認用に読み込めます。確認用読み込みでは、他人のクラウド保存を直接上書きしないようクラウドIDを引き継ぎません。
- 共有コードにはキャラクターデータと共有時のアカウント名が含まれます。秘密情報やGM向けメモを入力している場合は、共有前に内容を確認してください。
- Supabaseの `service_role` キーは使用しません。公開サイトに置くのはProject URLとPublishable/anon keyだけです。

## Tekeyで使う

- `values` はTekey APIの `addCharacter` でステータステーブル値を入れるためのパラメータです。
- `columns` はTekey APIの `setColumns` で部屋のカラムを揃えたい時に使えます。APIキー等を入力すると `setColumns` 用URLも生成します。
- APIキー、ルームID、ルームパスワードを入力すると `addCharacter` 用URLを生成します。これら3項目はローカル保存しません。
- チャットパレットはTekeyの `// パラメータ=値` と `###` プルダウン形式で出力します。
- 付属マニュアルに合わせ、ルームのシステムは `DiceBot` を想定しています。
- 付属のLoMTRPG用ステータステーブルとカウンターリモコンのJSONを、出力タブから保存できます。
- Tekey出力のステータス列は、付属ステータステーブルの `混乱 / HP / MP / 光 / PE / NE / 感情レベル / 狂気点 / ねじれ点 / パワー系 / 状態異常` に合わせています。

## ココフォリアで使う

- 「コマJSONをコピー」で、ココフォリアのClipboard API形式のキャラクターJSONをコピーできます。
- 盤面で貼り付けると、HP/MP/光などのステータス、能力値パラメータ、チャットパレット、メモが入ったコマとして作成できます。
- 固有パッシブと現在選択している選択パッシブの詳細効果をコマのメモ欄にも出力します。

## 実装済み

- 出身別能力値ダイス
- 得意分野による能力値補正とパッシブ表示、選択パッシブの効果表示
- 能力値ボーナス、HP、MP、回避力、防御力、命中力、速度、最大重量、光、狂気点上限の自動計算
- DEXボーナスごとの速度最低値/最大値の個別選択
- 名声点から階級・レベル算出
- 強化費用、装備費、所持重量、技能点の集計
- 武器・防具・装備品・義体の購入カタログと選択購入、アイテム詳細効果の表示
- 防具の斬/混斬/貫/混貫/打/混打耐性入力
- 光の種表、プロフィール、事務所欄
- 事務所の公式戦術スキル選択、SL別効果表示、Tekey/ココフォリア出力への反映
- フィクサー経歴表ロール
- 戦闘特技一覧表示、選択習得、必要技能点の自動加算
- 画面下部の技能点サマリー
- ブラウザ内アカウント、キャラクター保存庫、Supabaseクラウド保存、GM提出、管理者確認、共有コード
- PE/NE、感情レベル、混乱、Tekey状態異常欄
- Tekey向け values / columns / addCharacter URL / setColumns URL / チャットパレット出力
- Tekey用ステータステーブルJSON、カウンターリモコンJSON出力
- ココフォリア向けキャラクターClipboard API JSON出力
- ローカル保存、JSONインポート/エクスポート、テキスト出力、印刷

## メモ

購入カタログはルールブック記載の武器・防具耐性費用・アイテム・義体表を元にした公式表ベースの項目のみを入れています。防具は斬・混斬・貫・混貫・打・混打の各耐性を個別購入し、購入結果は「防具耐性セット」1行にまとめて反映されます。

GitHub Pagesなどで公開する場合、サイト内に装備名・数値・効果テキストを含むため、作者・配布元のガイドラインや二次利用条件を確認してください。

公開時の権利関係メモは `NOTICE.md` に分けています。必要に応じて、作者・配布元のガイドラインに合わせて文言を調整してください。
